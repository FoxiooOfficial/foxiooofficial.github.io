System requirements

Getting Started / System requirements (23 June, 2026)


Tails7 requires:

  • Clickteam Fusion 2.5 (preferably the latest version; you can check this via Help->About... in the Fusion window)
  • The Clickteam Fusion 2.5+ add-on
  • The project's Display Mode set to "Direct3D 11" (preferred) OR "Direct3D 9"
  • The latest version of the "Foxioo Shader Pack" (aka "FSP") must be installed.


Note

  • Do not change the Display Mode to "Direct3D 8" or "Standard"
    • Direct3D 8 is supported, but if you want to use it, it’s better to specify it in the startup parameters using "/D3D8"; the shaders used by Tails7 are too advanced and will not render in Direct3D 8 - only sprites will be displayed.
    • Standard mode may cause position transformation errors or "Memory Leaks" due to the large scale of objects that the CPU must calculate, consuming large amounts of RAM. Application crashes often occur; do not use this graphics mode.
  • If your graphics card does not support D3D11, the application will use D3D9; if it does not support D3D9, it will use D3D8; if it does not support D3D8, it will use Standard (CPU rendering).
  • If you want to compile games for other platforms, you must have purchased the DLCs that allow this.
  • On Android and iOS, there may be performance issues due to a large number of polygons or sprites.
