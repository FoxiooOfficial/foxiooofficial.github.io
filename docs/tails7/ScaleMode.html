Scale Mode

Objects / Tails7 Objects / Mapper Object / Scale Mode (23 June, 2026)


This behavior is a behavior responsible for scaling objects.

Different modes

Value

Description

Required version

"DEFAULT"

Scales object to fit the Tails7 environment (Default scaling)

R1.0.1_10.05.25

"DEFAULT_BASED_ON_X"

Scales object to fit the Tails7 environment but bases the scale for both axes on the resolution factor of the X-axis.

R1.0.2_29.11.25

"DEFAULT_BASED_ON_Y"

Scales object to fit the Tails7 environment but bases the scale for both axes on the resolution factor of the Y-axis.

"STATIC"

Scales the object to a fixed proportion, modified inversely by the cache override, ignoring resolution factor

R1.0.1_10.05.25

"RES"

Scales the object to raw scale values, modified only by resolution.

"FACTOR"

Scales the object to raw scale values, modified only by resolution factor.

"CONST"

Scales the object to the raw scale values (no modifications).

R1.0.2_29.11.25

"NONE"

No operations.

R1.0.1_10.05.25


Note

Setting a value other than those on the list (or "NONE")  will cause this behavior to be ignored.
