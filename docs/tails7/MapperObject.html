Mapper Object

Objects / Tails7 Objects / Mapper Object (23 June, 2026)


This object stores behavior rules for mapped objects, specifying how they should behave; you can add new "Modes" to it.


Starting with version R1.0.3, the function call order is explicit and is as follows:

    1. Distance Mode
    2. Layer Mode
    3. Mapping Mode
    4. Mapping Screen Mode
    5. Scale Mode
    6. Visible Mode
    7. Rotation Mode


This sequence is executed for all objects in Group.3D Objects on every frame once.

When a mapped object (Group.3D Objects) has the _Active flag set to False, it will not execute all instructions.

"Tails7MapperObject_DistanceMode"

Executes "Distance Mode"

"Sets the distance between the target position of the 3D.Object Group and the reference point (mainly the camera)."

"Tails7MapperObject_LayerMode"

Executes "Layer Mode"
"Layers each object individually to decide which should be on top and which underneath."

"Tails7MapperObject_MappingMode"

Executes "Mapping Mode"
"Calculations of the shader's inverse are applied to transform a given position through perspective to be adequate with the shader."

"Tails7MapperObject_MappingScreenMode"

Executes "Mapping Screen Mode"
"Displays the result of Mapping Mode but can also modify these results depending on the mode."

"Tails7MapperObject_ScaleMode"

Executes "Scale Mode"
"Scales the object to fit the calculations from Mapping Mode."

"Tails7MapperObject_VisibleMode"

Executes "Visible Mode"
"Determines when the object should be visible and when not."

"Tails7MapperObject_RotationMode"

Executes "Rotation Mode"
"Specifies how the object should be rotated."
