Objects / Tails7 Objects / Background Gradient / Internal (23 June, 2026)


This creates a gradient extending downwards from the horizon line.

It changes the texture to one containing a gradient, positions the object so it remains constantly on the horizon, and adjusts its rotation to match the camera's Y-axis rotation.


The layer's position is always set to _TargetZLayer( "Tails7 Info Object" );


All these operations are handled within the Mapping behavior of this object;
