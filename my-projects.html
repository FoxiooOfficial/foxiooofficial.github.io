My projects

My projects

Here you'll find a list of projects I'm currently working on or have worked on:

  • FSP - Foxioo Shader Pack

    FSP is a massive shader pack designed to work with Clickteam Fusion 2.5.

    It supports platforms such as: Windows (Direct3D9, Direct3D11, (full Direct3D8 support MAYBE coming soon)), Android, iOS, macOS, (HTML in the future).

    The shader pack currently includes over 420 effects!




  • Tails7

    An open-source framework/engine (idk what the difference is) called Mode 7 has been released for Clickteam Fusion 2.5+

    It features an object mapping system designed to create a 3D illusion.
    It also includes built-in polygons and the ability to switch to orthographic projection.
    It uses shaders (FSP), which allows for compilation on Windows, Android, iOS, and macOS.

  • FVEP - Foxioo's Vanilla Extension Plus!

    This extension is intended to replace the older "VanillaExt" version, which was only for testing purposes.

    FVEP is designed to introduce a scripting system using its own language, "VanillaScript," and to add a wide range of features not included by default in Fusion.

    It is currently unavailable.
