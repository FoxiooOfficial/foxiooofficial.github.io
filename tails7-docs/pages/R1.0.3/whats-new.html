What's new in Tails7 R1.0.3

27 March 2026

Learn about the new features in Tails7 R1.0.3 and find links to further documentation. This page has been updated for the latest release.

Tails7 R1.0.3, the successor to previous version R1.0.2. You can download R1.0.3 here.

R1.0.3_22.03.26

Release date: March 22, 2026 (Latest release)

Initial Features

  • There are now two types of polygons. The old polygons (which were included in versions R1.0.1 and R1.0.2) are now labeled "Quad"; the new polygons feature perspective corrections and do not disappear when they are too close. For more information, click here.
  • Calls to behavior calculations for engine objects are now explicit; the engine code contains a loop that calls each function individually.
  • Partial test support for Depth Buffer has been added (for Direct3D 9 only). For more information, click here.
  • Fixed the "TO_CAMERA_XYZ" Layer Mode; the Z position is now calculated correctly.
  • Entire Visible Mode has been fixed.
  • Background gradients objects have been updated to support camera positioning along the Z-axis for negative coordinates.

Important Note

Tails7 R1.0.3 requires Clickteam Fusion 2.5+ (Build R296.9 or newer). Make sure you have the latest version of Fusion installed before using Tails7.
