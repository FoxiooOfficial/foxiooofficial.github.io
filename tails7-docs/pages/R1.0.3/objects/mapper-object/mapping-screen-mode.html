Tails7 Mapper Object - Mapping Screen Mode

29 November 2025

Mapping Screen Mode is a behavior in the Tails7 Mapper Object to return the final screen position of objects from the 3D.Objects Group.

This behavior uses the values _FinalX and _FinalY, which are the final results from Mapping Mode.

Note

As of version R1.0.2_29.11.25, many modes have changed their names:

From "CLAMPX" to "CLAMP_X",

From "CLAMPY" to "CLAMP_Y",

From "INVERTEDX" to "INVERTED_X",

From "INVERTEDY" to "INVERTED_Y"

Different modes in Mapping Screen Mode

Mapping Screen Mode has its own 10 modes that every object from the 3D.Objects Group can use to behave individually.  The string variable _MappingScreenMode is assigned a value that defines the given mode.

Value Description Date Added
"DEFAULT" Returns _FinalX and _FinalY from 3D.Objects Group directly to the object's position on the X and Y axes. R1.0.1_10.05.25
"DEFAULT_ONLY_X" Returns _FinalX from 3D.Objects Group directly to the object's position on the X axis. R1.0.2_29.11.25
"DEFAULT_ONLY_Y" Returns _FinalY from 3D.Objects Group directly to the object's position on the Y axis. R1.0.2_29.11.25
"CLAMP" Returns _FinalX and _FinalY from 3D.Objects Group and modifies it to clamp the value from 0 to Frame Width for X and from 0 to Frame Height for Y, then returns it to the object on the X and Y axes. R1.0.1_10.05.25
"CLAMP_X" Returns _FinalX and _FinalY from 3D.Objects Group and modifies _FinalX to clamp the value from 0 to Frame Width , then returns it to the object on the X and Y axes. R1.0.1_10.05.25
"CLAMP_Y" Returns _FinalX and _FinalY from 3D.Objects Group and modifies _FinalY to clamp the value from 0 to Frame Height , then returns it to the object on the X and Y axes. R1.0.1_10.05.25
"INVERTED" Returns _FinalX and _FinalY from 3D.Objects Group and modifies the values by inverting them from the center of the screen, i.e. Frame Width / 2.0 - _FinalX for X, and for Y, returns Frame Height / 2.0 - _FinalY , then returns it to the object on the X and Y axes. R1.0.1_10.05.25
"INVERTED_X" Returns _FinalX and _FinalY from 3D.Objects Group and modifies only X which is inverted from the center of the screen, i.e. Frame Width / 2.0 - _FinalX , then returns it to the object on the X and Y axes. R1.0.1_10.05.25
"INVERTED_Y" Returns _FinalX and _FinalY from 3D.Objects Group and modifies only Y which is inverted from the center of the screen, i.e. Frame Height / 2.0 - _FinalY , then returns it to the object on the X and Y axes. R1.0.1_10.05.25
"PSX" Returns _FinalX and _FinalY from 3D.Objects Group and modifies them to simulate the characteristic PSX/PlayStation 1 style of object or polygon placement on the screen, then returns it to the object on the X and Y axes. R1.0.2_29.11.25

Note

Setting a value other than those on the list will cause this behavior to be ignored.
