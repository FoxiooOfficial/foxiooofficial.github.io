Tails7 Mapper Object - Scale Mode

29 November 2025

Scale Mode in the Tails7 Mapper Object is a behavior responsible for scaling objects in the 3D.Objects Group.

Different modes in Scale Mode

Scale Mode has its own 7 modes that every object from the 3D.Objects Group can use to behave individually.  The string variable _ScaleMode is assigned a value that defines the given mode.

Value Description Date Added
"DEFAULT" The object scaling mode which scales the object in the 3D.Objects Group to fit the space in the shader. R1.0.1_10.05.25
"DEFAULT_BASED_ON_X" The object scaling mode which scales the object in the 3D.Objects Group to fit the space in the shader but bases the scale for both axes on the resolution factor of the X-axis. R1.0.2_29.11.25
"DEFAULT_BASED_ON_Y" The object scaling mode which scales the object in the 3D.Objects Group to fit the space in the shader but bases the scale for both axes on the resolution factor of the Y-axis. R1.0.2_29.11.25
"STATIC" The object scaling mode which scales the object in the 3D.Objects Group to be constant. R1.0.1_10.05.25
"RES" The object scaling mode which scales the object in the 3D.Objects Group to be constant but depends on the project resolution. R1.0.1_10.05.25
"FACTOR" The object scaling mode which scales the object in the 3D.Objects Group to be constant but depends on the project resolution factor. R1.0.1_10.05.25
"CONST" The scaling mode which directly returns _ScaleX * _Scale for the X-axis scale and _ScaleY * _Scale for the Y-axis scale for objects from the 3D.Objects Group. R1.0.2_29.11.25

Note

Setting a value other than those on the list will cause this behavior to be ignored.
