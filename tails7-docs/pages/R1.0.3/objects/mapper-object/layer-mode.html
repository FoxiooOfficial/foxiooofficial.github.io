Tails7 Mapper Object - Layer Mode

29 November 2025

Layer Mode in the Tails7 Mapper Object is responsible for the layer ordering of objects from the 3D.Objects Group.

Note

As of version R1.0.2_29.11.25, many modes have changed their names:

From "TOCAMERA" to "TO_CAMERA_XY",

From "TOCAMERAZ" to "TO_PLAYER_XYZ",

From "TOPLAYER" to "TO_PLAYER_XY",

From "TOPLAYERZ" to "TO_PLAYER_XYZ",

Different modes in Layer Mode

Layer Mode has 4 different modes that every object from the 3D.Objects Group can use to behave individually.  In the string variable _RotationMode, we assign a value that defines the given mode.

Value Description Date Added
"TO_CAMERA_XY" Takes into account the distance between the 3D.Objects Group object and the Tails7 Camera Object on the X and Y axes. R1.0.1_10.05.25
"TO_CAMERA_XYZ" Takes into account the distance between the 3D.Objects Group object and the Tails7 Camera Object on the X, Y, and Z axes. R1.0.1_13.05.25
"STATIC" Takes into account the distance from coordinates 0; 0 to _PositionX and _PositionY of the 3D.Objects Group object. R1.0.1_10.05.25
"TO_PLAYER_XY" Takes into account the distance between the 3D.Objects Group object and the Player Hitbox on the X and Y axes. R1.0.1_10.05.25
"TO_PLAYER_XYZ" Takes into account the distance between the 3D.Objects Group object and the Player Hitbox on the X, Y, and Z axes. R1.0.1_13.05.25

Note

Setting a value other than those on the list will cause this behavior to be ignored.
