Tails7 Mapper Object - Rotation Mode

29 November 2025

Rotation Mode in the Tails7 Mapper Object is a behavior responsible for the rotation of objects from the 3D.Objects Group.

This mode uses two variables: _RotationBase and _RotationMod.

_RotationBase is automatically calculated by the mode, and _RotationMod is a value that is added to the final rotation result. For example, if you want to add 5 degrees to the rotation, you assign 5 to this variable.

Different modes in Rotation Mode

Rotation Mode has a single mode that every object from the 3D.Objects Group can use to behave individually.  The string variable _RotationMode is assigned a value that defines the given mode.

Value Description Date Added
"DEFAULT" Changes the object's rotation dependent on the camera's Y-axis rotation for objects from the 3D.Objects Group. R1.0.2_18.05.25

Note

Setting a value other than those on the list will cause this behavior to be ignored.
