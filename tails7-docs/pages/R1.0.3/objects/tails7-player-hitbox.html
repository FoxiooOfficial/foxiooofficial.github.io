Tails7 Player Hitbox

29 November 2025

Tails7 Player Hitbox is an object that doesn't play a major role in the engine.
Its purpose is to serve as a ready-to-use object for player hitbox for checking collisions with, for example, walls.

The entire movement system of this object is in the Position behavior of the Tails7 Camera Object

The object is also used in Layer Mode in the Tails7 Mapper Object for several modes.

Note

Since version R1.0.2_29.11.25 this object has changed its name from Player Hitbox to Tails7 Player Hitbox.

This aims to prevent situations where you have an object with the same name in your project and you want to paste the Tails7 base into your project, so that Fusion correctly interprets that these are two separate objects.
(I once had such a situation while making my game...)

Variables in Tails7 Player Hitbox

This object has only one variable in the Alterable Values section

Variable Default value Description Date added
_Speed 15 This variable specifies the movement speed in the built-in movement system in the Position behavior located in the Tails7 Camera Object. R1.0.1_10.05.25
