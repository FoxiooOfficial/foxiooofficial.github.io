Polygon Objects

29 November 2025

Polygons are objects that use shaders from the Quad family or similar ones that transform 4 points to modify the shape, creating a 3D illusion.

These objects mainly use 4 separate Sprites for points A, B, C, D.



Each point has its own order, starting from the top-left corner (A (0; 0)), then top-right (B (1; 0)), then bottom-right (C (1; 1)), and finally bottom-left (D (0; 1)).

Polygon also has two behaviors, Startup and Set Points. The first behavior initializes the polygon and expands it to fill the screen; at this stage the polygon can be scaled, which is noted below. The second behavior Set Points sets points A, B, C, D at appropriate positions on the screen and sets the polygon's visibility.

Note

Polygons are not perfect and may disappear when you are too close, or at direct camera angles there may be visual artifacts, which is the fault of the shaders that I partially fixed.


Polygons are not created by default when the graphics API is Direct3D8, CPU or DirectDraw!
