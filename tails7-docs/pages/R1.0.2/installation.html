Installing Tails7 R1.0.2

29 November 2025

Here you will learn how to properly install the project.

Note

Before installing the engine, make sure you have:

  1. Clickteam Fusion 2.5 (Build 296.7 or newer!)
  2. Clickteam Fusion 2.5+ DLC
  3. "FSP", i.e. "Foxioo Shader Pack" (also included in this guide)

Guide

1. W przeglądarce, przejdź na stronę GitHub silnika Tails7.






2. Pobierz projekt klikając Code -> Download ZIP






3. Wybierz folder gdzie chcesz pobrać cały plik zip.






4. Otwórz folder gdzie znajduje się plik zip z Tails7.






5. Wyodrębnij plik zip z Tails7.






6. W przeglądarce, przejdź teraz na stronę GitHub paczki FSP.






7. Również kliknij w Code -> Download ZIP by pobrać shader pack który jest potrzebny do Tails7.






8. W Clickteam Fusion przejdź do Tools -> Windows Explorer, powinno się otworzyć okno pokazując gdzie Fusion jest zainstalowany.






9. Przenieś folder Foxioo Shader Pack w podfolderze w zip do folderu Effects gdzie Fusion jest zainstalowany i uruchom pownownie Fusion.






10. Wróć do folderu gdzie pobrałeś projekt i otwórz Tails7.mfa.






11. Gdy Tails7.mfa się otworzy w Fusion, uruchom Runtime.






12. Gratulacje! Zainstalowałeś poprawnie silnik i możesz zacząć pracę nad twoim projektem/grą!
