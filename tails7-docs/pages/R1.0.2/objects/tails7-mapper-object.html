Tails7 Mapper Object

29 November 2025

Tails7 Mapper Object jest odpowiedzialny za takie operacje jak: ustawianie pozycji, skalowanie, obracanie, warstwowanie dla WSZYSTKICH obiektów 3D.Objects Group gdy są aktywne.

Dzieje się tak gdyż obiekt ten ma w sobie kilka zachowań które zachowują pewną ciągłość.

Nazywamy to "Behavior chain mapping for 3D objects.Objects Group executable by Tails7 Mapper Object" (Bardzo długa nazwa, wiem), albo poprostu "łańcuchem".


Kolejność łańcucha

Sekwencja łańcucha i kolejność następujących zachowań:

  1. Distance Mode - Ustawia dystans targetowej pozycji 3D.Object Group a punktem odniesienia (Głównie kamerą).
  2. Mapping Mode - Stosowane są kalkulacje odwrotności shadera by przekształcić daną pozycję przez perspektywę by była adekwatna z shaderem.
  3. Mapping Screen Mode - Wyświetla wynik Mapping Mode jednak może też te wyniki zmodyfikować zależnie od trybu.
  4. Scale Mode - Skaluje obiekt by dopasować go do kalkulacji z Mapping Mode.
  5. Layer Mode - Warstwuje każdy obiekt indywidualnie tak by zdecydować który ma być na wieszchu a który podspodem.
  6. Visible Mode - Określa kiedy obiekt ma być widoczny a kiedy nie.
  7. Rotation Mode - Obraca każdy obiekt.
