Tails7 Mapper Object - Distance Mode

29 November 2025

Distance Mode w obiekcie Tails7 Mapper Object jako zachowanie wykonuje się jako pierwszę wględem łańcucha.

Zachowanie to oblicza dystans który będzie potrzebny do obliczeń zachowania Mapping Mode.

Różne tryby w Distance Mode

Distance Mode ma swoje własne 2 tryby które każdy obiekt z grupy 3D.Objects Group może indywidualnie się zachowywać.  W zmiennej string _DistanceMode przypisujemy do niej wartość która określa dany tryb.

Wartość Opis Data dodania
"DEFAULT" Ustawia wartości _DistanceX i _DistanceY na różnice _PositionX i _PositionY obiektu 3D.Objects Group pomiędzy pozycją obiektu Tails7 Camera Object w osi X i Y. R1.0.1_10.05.25
"STATIC" Ustawia warości _DistanceX i _DistanceY na _PositionX i _PositionY. Dystans w tym przypadku jest stały. R1.0.1_10.05.25

Note

Ustawienie wartości innej niż z listy spowoduje ignorowanie tego zachowania
