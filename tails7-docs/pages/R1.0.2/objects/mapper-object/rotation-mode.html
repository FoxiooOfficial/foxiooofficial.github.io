Tails7 Mapper Object - Rotation Mode

29 November 2025

Rotation Mode w obiekcie Tails7 Mapper Object jako zachowanie jest odpowiadanie za obrót obiektów z grupy 3D.Objects Group.

Tryb ten wykorzystuje dwie zmienne: _RotationBase i _RotationMod.

_RotationBase jest automatycznie obliczany przez tryb a _RotationMod to wartość która do wyniku końcowego obrotu jest dodawana, przykładowo jak chcesz by do rotation dodać 5 stopni, to do tej zmiennej przypisujesz 5.

Różne tryby w Rotation Mode

Rotation Mode ma swój jeden tryb który każdy obiekt z grupy 3D.Objects Group może indywidualnie się zachowywać.  W zmiennej string _RotationMode przypisujemy do niej wartość która określa dany tryb.

Wartość Opis Data dodania
"DEFAULT" Zmienia obrót obiektu zależny od obrotu kamery w osi Y dla obiektów z grupy 3D.Objects Group. R1.0.2_18.05.25

Note

Ustawienie wartości innej niż z listy spowoduje ignorowanie tego zachowania
