Tails7 Mapper Object - Layer Mode

29 November 2025

Layer Mode w obiekcie Tails7 Mapper Object jako zachowanie jest odpowiadanie za kolejność warstwowania obiektów z grupy 3D.Objects Group.

Note

Od wersji R1.0.2_29.11.25 wiele trybów zmieniło nazwę:

Z "TOCAMERA" na "TO_CAMERA_XY",

Z "TOCAMERAZ" na "TO_PLAYER_XYZ",

Z "TOPLAYER" na "TO_PLAYER_XY",

Z "TOPLAYERZ" na "TO_PLAYER_XYZ",

Różne tryby w Layer Mode

Layer Mode ma swoje 4 różne tryby które każdy obiekt z grupy 3D.Objects Group może indywidualnie się zachowywać.  W zmiennej string _RotationMode przypisujemy do niej wartość która określa dany tryb.

Wartość Opis Data dodania
"TO_CAMERA_XY" Bierze pod uwagę dystans między obiektem 3D.Objects Group a Tails7 Camera Object w osiach X i Y. R1.0.1_10.05.25
"TO_CAMERA_XYZ" Bierze pod uwagę dystans między obiektem 3D.Objects Group a Tails7 Camera Object w osiach X, Y i Z. R1.0.1_13.05.25
"STATIC" Bierze pod uwagę dystans od kordynatów 0; 0 a _PositionX i _PositionY z obiektu 3D.Objects Group. R1.0.1_10.05.25
"TO_PLAYER_XY" Bierze pod uwagę dystans między obiektem 3D.Objects Group a Player Hitbox w osiach X i Y. R1.0.1_10.05.25
"TO_PLAYER_XYZ" Bierze pod uwagę dystans między obiektem 3D.Objects Group a Player Hitbox w osiach X, Y i Z. R1.0.1_13.05.25

Note

Ustawienie wartości innej niż z listy spowoduje ignorowanie tego zachowania
