Tails7 Render Ground

29 November 2025

Tails7 Render Ground jest to obiekt który rysuje perspektwyę dając efekt przestrzeni przez nałożony shader. Mapowanie obiektów są bazowane na obliczeniach z shadera, tak by dopasować symulowanie trójwymiaru przez prawidłowo ustawienie obiektów na ekranie.

Obiekt ten jest tworzony przez zachowanie na początku w Startup Set Render przez obiekt Tails7 Info Object.
Sprawdza minimalną rozdzielczość ramki i dopasowywuje się tak by wypełnić cały ekran przez duplikowanie się obiektu. Wszystko dobrze działą gdy szerokość jest większa niż wysokość ramki, mogą być problemy z rozdzielczoścami które są w pionie.

Note

Obiekt ten nie są domyślnie wyświetlany gdy API graficzne jest Direct3D8, CPU albo DirectDraw z powodu braku kompatybilności/wyświetlania shaderów w tych trybach.

Zmienne w shaderze Mode 7

Zmienne tyczą wersji 1.9 (08.11.2025) shadera.

Zmienna Domyślna wartość Opis
_PosX 0.0 A variable that determines the position of the texture in the X axis.
_PosY 0.0 A variable that determines the position of the texture in the Y axis.
_PosZ 1.0 A variable that specifies the position of the texture in the Z-axis.
_RotX 0.0 A variable that determines rotation in the X axis.
_RotY 0.0 A variable that determines rotation in the Y axis.
_RotZ 0.0 *The variable that determines rotation in the Z-axis. (This is otherwise known as the displacement of the point of the center of the perspective in the Y axis).
_RotXPointX 0.0 A variable that determines point position in the X axis for Rotation X.
_RotXPointY 0.0 A variable that determines point position in the Y axis for Rotation X.
_RotYPointX 0.0 A variable that determines point position in the X axis for Rotation Y.
_RotXPointY 0.0 A variable that determines point position in the Y axis for Rotation Y.
_OffsetX 0.0 A variable that determines the shift of the perspective center point in the X axis.
_Distortion 1.0 A variable specifying how much the texture should be curved to give perspective immediately. (0 is no curvature, 1 is the default value giving perspective effect).
_Scale 1.0 This variable determines the overall scale of the texture (in X and Y axes).
_ScaleX 1.0 This variable determines the scale of the texture in the X axis.
_ScaleY 1.0 This variable determines the scale of the texture in the Y axis.
_PosOffsetX 0.0 This variable sets the offset of the texture position in the X axis. (For the _ScaleX and _Scale variable).
_PosOffsetY 0.0 This variable sets the offset of the texture position in the Y axis. (For the _ScaleY and _Scale variable).
_Looping_Mode 0 This variable determines how the texture is repeated:
REPEAT (repeats the texture image);
MIRRORED (repeats the texture image, but mirrors it with each repetition);
CLAMP (extends the texture beyond coordinates 0 and 1);
BORDER (the texture will be invisible after exceeding coordinates 0 and 1).

For more information, visit: https://learnopengl.com/Getting-started/Textures

List of modes for separately (X, Y) by _Looping_Mode:
0: (REPEAT, REPEAT);
1: (MIRRORED, MIRRORED);
2: (CLAMP, CLAMP);
3: (BORDER, BORDER);

4: (MIRRORED, REPEAT);
5: (CLAMP, REPEAT);
6: (BORDER, REPEAT);

7: (REPEAT, MIRRORED);
8: (CLAMP, MIRRORED);
9: (BORDER, MIRRORED);

10: (REPEAT, CLAMP);
11: (MIRRORED, CLAMP);
12: (BORDER, CLAMP);

13: (REPEAT, BORDER);
14: (MIRRORED, BORDER);
15: (CLAMP, BORDER) Please note that _Looping_Mode from 0 to 15 only works for Direct3D11 and OpenGL (Android, iOS and Mac); For Direct3D9, only 0 to 3 are available.
_Render_Sky 0 This variable determines whether the "sky" should be visible. (This is a reflection of the floor with the axes reversed.)
_Blending_Mode 0 This variable determines what the base should be. Setting it to 0 sets the base as the texture of the object on which the effect is applied, and setting it to 1 sets this as the background.
_Is_Pre_296_Build 0 (Only applies to Android and iOS!) This variable determines whether to use the newer background formatting. In Clickteam Fusion 2.5 for build 295.10 and earlier, the background texture was flipped along the Y axis, but this was changed in build 296.1. Check this option if you are using an older build!

Zmienne w Tails7 Render Ground

Obiekt ten ma tylko dwie zmienne w sekcji Alterable Values

Zmienna Domyślna wartość Opis Data dodania
_RenderOffsetX (Automatically) Zmienna ta podczas tworzenia obiektu określa jaki ma być offset renderu w osi X. R1.0.1_10.05.25
_RenderOffsetY (Automatically) Zmienna ta podczas tworzenia obiektu określa jaki ma być offset renderu w osi Y. R1.0.1_10.05.25

Obiekt ten ma tylko dwie zmienne w sekcji Flags

Zmienna Domyślna wartość Opis Data dodania
_Active 0 Zmienna ta gdy jest ustawiona na 0, roszerza obiekt by dopasować jego skalę, następnie ustawia tą zmienną na 1 uznając że obiekt ten jest gotowy. R1.0.1_10.05.25
