Tails7 Background Gradient - External

29 November 2025

The Tails7 External Background Gradient object creates a gradient extending upwards from the horizon line. It changes the texture to one containing a gradient, positions the object so it remains constantly on the horizon, and adjusts its rotation to match the camera's Y-axis rotation.

Its layer position is equal to _TargetZLayer( "Tails7 Info Object" ) - 1, meaning it will target a layer one lower than the Tails7 Render Ground object.

All these operations are handled within the **Mapping** behavior of this object.

Variables in this object

This object has two variables in the Alterable Variables section

Variable Default Value Description Date Added
_Size 2.0 This variable determines the height of the gradient on the Y-axis. R1.0.1_10.05.25
_Calc (Automatically) This variable is used for temporary calculations for the Mapping behavior to correctly position the object. R1.0.1_10.05.25

Note

This object has had several revisions across versions:
As of version R1.0.2_18.05.25, the object was rewritten to align with the camera's Y-axis rotation.
As of version R1.0.2_29.11.25, a dependency on the variables _RotYPointX and _RotYPointY of the Tails7 Info Object was added.
