Tails7 Info Object

29 November 2025

Tails7 Info Object jest to obiekt który przechowuje podstawowe informację w zmiennych, głównie dla Tails7 Ground Render.

Obiekt jest również wykorzystywany w zachowaniach w obiekcie Tails7 Mapper Object.

Zmienne w Tails7 Info Object

Obiekt ten ma tylko jedną zmienną w sekcji Alterable Values

Zmienna Domyślna wartość Opis Data dodania
_WorldSize 1 Zmienna ta określa poziom zagęszczenia świata, domyślnie jest ustawiona na 1, jeśli przykładowo ustawimy na 2 to dystans poruszania się, jak i obiektów będzie 2 razy mniejsza.
(Nie zalecam zmieniać tej zmiennej) 		R1.0.1_10.05.25
_TargetZLayer 2 Zmienna ta określa na której warstwie ma zostać stworzony obiekt Tails7 Render Ground. R1.0.1_10.05.25
_PosX (Automatically) Zmienna zwraca informację o pozycji X obiektu Tails7 Camera Object. R1.0.1_10.05.25
_PosY (Automatically) Zmienna zwraca informację o pozycji Y obiektu Tails7 Camera Object. R1.0.1_10.05.25
_PosZ 1.0 Zmienna zawiera informację o pozycji Z oraz może jednocześnie ustawiać tą pozycję Z kamery. R1.0.1_10.05.25
_RotX 0.0 Zmienna zawiera informację o rotacji w osi X oraz może jednocześnie ustawiać tą rotację X kamery. R1.0.1_10.05.25
_RotY 0.0 Zmienna zawiera informację o rotacji w osi Y oraz może jednocześnie ustawiać tą rotację Y kamery. R1.0.1_10.05.25
_RotZ 0.0 Zmienna zawiera informację o rotacji w osi Z oraz może jednocześnie ustawiać tą rotację Z kamery. R1.0.1_10.05.25
_Distortion 1.0 Zmienna zawiera informację o stopniu distortion oraz może jednocześnie ustawiać ten distortion kamery. R1.0.1_10.05.25
_RotXPointX 0.0 Zmienna zawiera informację o przesunięciu punktu rotacji X w osi X oraz może jednocześnie ustawiać tą wartość. R1.0.1_10.05.25
_RotXPointY 0.0 Zmienna zawiera informację o przesunięciu punktu rotacji X w osi Y oraz może jednocześnie ustawiać tą wartość. R1.0.1_10.05.25
_RotYPointX 0.0 Zmienna zawiera informację o przesunięciu punktu rotacji Y w osi X oraz może jednocześnie ustawiać tą wartość. R1.0.1_10.05.25
_RotYPointY 0.0 Zmienna zawiera informację o przesunięciu punktu rotacji Y w osi Y oraz może jednocześnie ustawiać tą wartość. R1.0.1_10.05.25
_Scale 1.0 Zmienna zawiera informację o skali tekstury ground w obu osiach oraz może jednocześnie ustawiać tą wartość. R1.0.1_10.05.25
_ScaleX 1.0 Zmienna zawiera informację o skali tekstury ground w osi X oraz może jednocześnie ustawiać tą wartość. R1.0.1_10.05.25
_ScaleY 1.0 Zmienna zawiera informację o skali tekstury ground w osi Y oraz może jednocześnie ustawiać tą wartość. R1.0.1_10.05.25
_PosOffsetX 0.0 Zmienna zawiera informację o przesunięciu tekstury w osi X (podawana w wartościach texCoordowych czyli od 0 do 1) oraz może jednocześnie ustawiać tą wartość. R1.0.1_10.05.25
_PosOffsetY 0.0 Zmienna zawiera informację o przesunięciu tekstury w osi Y (podawana w wartościach texCoordowych czyli od 0 do 1) oraz może jednocześnie ustawiać tą wartość. R1.0.1_10.05.25
_TextureID 1 Zmienna zawiera informację jaki frame tekstury ground ma być użyty podczas tworzenia obiektu Tails7 Ground Render oraz może jednocześnie ustawiać tą wartość. R1.0.1_10.05.25
_TextureSizeX 2048 Zmienna zawiera informację o skali tekstury ground w osi X, podobnie jak _ScaleX jednak tu używa się jednostki pikseli oraz może jednocześnie ustawiać tą wartość. R1.0.1_10.05.25
_TextureSizeY 2048 Zmienna zawiera informację o skali tekstury ground w osi Y, podobnie jak _ScaleY jednak tu używa się jednostki pikseli oraz może jednocześnie ustawiać tą wartość. R1.0.1_10.05.25
_ResolutionX 1280 Zmmienna ta określa jaki targetową rozdzielczość w osi X planujesz, ma to znaczenie dla obliczeń które używają chociażby factoru. R1.0.1_10.05.25
_ResolutionY 720 Zmmienna ta określa jaki targetową rozdzielczość w osi Y planujesz, ma to znaczenie dla obliczeń które używają chociażby factoru. R1.0.1_10.05.25
_RenderMinimumSize (Automatically) Zmienna ta zaiwera informacje o najmniejszej wartości z pośród _ResolutionX i _ResolutionY. R1.0.1_10.05.25
_RenderMiddleSizeX (Automatically) Zmienna ta zwraca Frame Width / 2.0. R1.0.1_10.05.25
_RenderMiddleSizeY (Automatically) Zmienna ta zwraca Frame Height / 2.0. R1.0.1_10.05.25
_ScaleFactorX (Automatically) Zmienna ta zwraca Frame Width / ( _ResolutionX( "Tails7 Info Object" ) + 0.0 ). R1.0.1_10.05.25
_ScaleFactorY (Automatically) Zmienna ta zwraca Frame Height / ( _ResolutionY( "Tails7 Info Object" ) + 0.0 ). R1.0.1_10.05.25
_WidthRatio (Automatically) Zmienna ta zwraca Frame Width / 320.0. R1.0.1_10.05.25
_AspectCorrection (Automatically) Zmienna ta zwraca 1.25 - ( _WidthRatio( "Tails7 Info Object" ) - 1.0 ) * 0.3. R1.0.1_10.05.25
