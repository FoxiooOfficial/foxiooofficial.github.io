Tails7 Info Object

29 November 2025

Tails7 Info Object is an object that stores basic information in variables, mainly for Tails7 Ground Render.

The object is also used in behaviors in the Tails7 Mapper Object.

Variables in Tails7 Info Object

This object has only one variable in the Alterable Values section

Variable Default value Description Date added
_WorldSize 1 This variable specifies the world density level, by default it is set to 1, if for example we set it to 2 then the movement distance, as well as objects, will be 2 times smaller.
(I don't recommend changing this variable) 		R1.0.1_10.05.25
_TargetZLayer 2 This variable specifies on which layer the Tails7 Render Ground object should be created. R1.0.1_10.05.25
_PosX (Automatically) Variable returns information about the X position of the Tails7 Camera Object. R1.0.1_10.05.25
_PosY (Automatically) Variable returns information about the Y position of the Tails7 Camera Object. R1.0.1_10.05.25
_PosZ 1.0 Variable contains information about the Z position and can simultaneously set this Z position of the camera. R1.0.1_10.05.25
_RotX 0.0 Variable contains information about the rotation on the X axis and can simultaneously set this X rotation of the camera. R1.0.1_10.05.25
_RotY 0.0 Variable contains information about the rotation on the Y axis and can simultaneously set this Y rotation of the camera. R1.0.1_10.05.25
_RotZ 0.0 Variable contains information about the rotation on the Z axis and can simultaneously set this Z rotation of the camera. R1.0.1_10.05.25
_Distortion 1.0 Variable contains information about the distortion level and can simultaneously set this distortion of the camera. R1.0.1_10.05.25
_RotXPointX 0.0 Variable contains information about the X rotation point offset on the X axis and can simultaneously set this value. R1.0.1_10.05.25
_RotXPointY 0.0 Variable contains information about the X rotation point offset on the Y axis and can simultaneously set this value. R1.0.1_10.05.25
_RotYPointX 0.0 Variable contains information about the Y rotation point offset on the X axis and can simultaneously set this value. R1.0.1_10.05.25
_RotYPointY 0.0 Variable contains information about the Y rotation point offset on the Y axis and can simultaneously set this value. R1.0.1_10.05.25
_Scale 1.0 Variable contains information about the ground texture scale on both axes and can simultaneously set this value. R1.0.1_10.05.25
_ScaleX 1.0 Variable contains information about the ground texture scale on the X axis and can simultaneously set this value. R1.0.1_10.05.25
_ScaleY 1.0 Variable contains information about the ground texture scale on the Y axis and can simultaneously set this value. R1.0.1_10.05.25
_PosOffsetX 0.0 Variable contains information about the texture offset on the X axis (given in texCoord values i.e., from 0 to 1) and can simultaneously set this value. R1.0.1_10.05.25
_PosOffsetY 0.0 Variable contains information about the texture offset on the Y axis (given in texCoord values i.e., from 0 to 1) and can simultaneously set this value. R1.0.1_10.05.25
_TextureID 1 Variable contains information about which ground texture frame should be used when creating the Tails7 Ground Render object and can simultaneously set this value. R1.0.1_10.05.25
_TextureSizeX 2048 Variable contains information about the ground texture scale on the X axis, similar to _ScaleX but here pixel units are used and can simultaneously set this value. R1.0.1_10.05.25
_TextureSizeY 2048 Variable contains information about the ground texture scale on the Y axis, similar to _ScaleY but here pixel units are used and can simultaneously set this value. R1.0.1_10.05.25
_ResolutionX 1280 This variable specifies what target resolution you plan for the X axis, it matters for calculations that use, for example, the factor. R1.0.1_10.05.25
_ResolutionY 720 This variable specifies what target resolution you plan for the Y axis, it matters for calculations that use, for example, the factor. R1.0.1_10.05.25
_RenderMinimumSize (Automatically) This variable contains information about the smallest value among _ResolutionX and _ResolutionY. R1.0.1_10.05.25
_RenderMiddleSizeX (Automatically) This variable returns Frame Width / 2.0. R1.0.1_10.05.25
_RenderMiddleSizeY (Automatically) This variable returns Frame Height / 2.0. R1.0.1_10.05.25
_ScaleFactorX (Automatically) This variable returns Frame Width / ( _ResolutionX( "Tails7 Info Object" ) + 0.0 ). R1.0.1_10.05.25
_ScaleFactorY (Automatically) This variable returns Frame Height / ( _ResolutionY( "Tails7 Info Object" ) + 0.0 ). R1.0.1_10.05.25
_WidthRatio (Automatically) This variable returns Frame Width / 320.0. R1.0.1_10.05.25
_AspectCorrection (Automatically) This variable returns 1.25 - ( _WidthRatio( "Tails7 Info Object" ) - 1.0 ) * 0.3. R1.0.1_10.05.25

This object has only one variable in the Alterable Strings section

Variable Default value Description Date added
_Version R1.0.1_13.05.25 This variable informs which version of Tails7 you are currently using. R1.0.1_10.05.25
_API (Automatically) Variable informs which graphics API the game is currently using if the project runs on Windows. The text may contain: Direct3D8, Direct3D9, DirectD3D11, DirectDraw or CPU R1.0.1_10.05.25

This object has only one variable in the Flags section

Variable Default value Description Date added
_Update 0 This variable activates the creation of Tails7 Ground Render objects from scratch. It is useful when you want to change the ground object texture if the texture has a different resolution. R1.0.1_10.05.25
_IsUpdating (Automatically) Temporary variable that informs about the status of creating Tails7 Ground Render objects when the _Update flag is activated. R1.0.1_10.05.25
