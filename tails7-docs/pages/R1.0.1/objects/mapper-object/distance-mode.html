Tails7 Mapper Object - Distance Mode

29 November 2025

Distance Mode in the Tails7 Mapper Object executes first in the chain.

This behavior calculates the distance that will be required for the calculations of Mapping Mode.

Different modes in Distance Mode

Distance Mode has its own 2 modes that every object from the 3D.Objects Group can use to behave individually.  In the string variable _DistanceMode, we assign a value that defines the given mode.

Value Description Date Added
"DEFAULT" Sets the values of _DistanceX and _DistanceY to the difference between _PositionX and _PositionY of the 3D.Objects Group object and the position of the Tails7 Camera Object on the X and Y axes. R1.0.1_10.05.25
"STATIC" Sets the values of _DistanceX and _DistanceY to _PositionX and _PositionY. The distance is constant in this case. R1.0.1_10.05.25

Note

Setting a value other than those on the list will cause this behavior to be ignored.
