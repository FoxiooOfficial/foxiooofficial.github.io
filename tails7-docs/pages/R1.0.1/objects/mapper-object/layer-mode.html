Tails7 Mapper Object - Layer Mode

29 November 2025

Layer Mode in the Tails7 Mapper Object is responsible for the layer ordering of objects from the 3D.Objects Group.

Different modes in Layer Mode

Layer Mode has 4 different modes that every object from the 3D.Objects Group can use to behave individually.  In the string variable _RotationMode, we assign a value that defines the given mode.

Value Description Date Added
"TOCAMERA" Takes into account the distance between the 3D.Objects Group object and the Tails7 Camera Object on the X and Y axes. R1.0.1_10.05.25
"TOCAMERAZ" Takes into account the distance between the 3D.Objects Group object and the Tails7 Camera Object on the X, Y, and Z axes. R1.0.1_13.05.25
"STATIC" Takes into account the distance from coordinates 0; 0 to _PositionX and _PositionY of the 3D.Objects Group object. R1.0.1_10.05.25
"TOPLAYER" Takes into account the distance between the 3D.Objects Group object and the Player Hitbox on the X and Y axes. R1.0.1_10.05.25
"TOPLAYERZ" Takes into account the distance between the 3D.Objects Group object and the Player Hitbox on the X, Y, and Z axes. R1.0.1_13.05.25

Note

Setting a value other than those on the list will cause this behavior to be ignored.
