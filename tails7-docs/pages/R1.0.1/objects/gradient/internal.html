Tails7 Background Gradient - Internal

21 December 2025

The Tails7 Internal Background Gradient object creates a gradient extending downwards from the horizon line. It changes the texture to one containing a gradient, positions the object so it remains constantly on the horizon, and adjusts its rotation to match the camera's Y-axis rotation (and 180° is added to the rotation).

Additionally, the camera's Z position is also taken into account for the object's scale.

Its layer position is equal to _TargetZLayer( "Tails7 Info Object" ), meaning it will target the same layer as the Tails7 Render Ground object, but it will always be displayed above it.

All these operations are handled within the Mapping behavior of this object.

Variables in this object

This object has two variables in the Alterable Variables section

Variable Default Value Description Date Added
_Size 0.5 This variable determines the height of the gradient on the Y-axis. R1.0.1_10.05.25
_Calc (Automatically) This variable is used for temporary calculations for the Mapping behavior to correctly position the object. R1.0.1_10.05.25
