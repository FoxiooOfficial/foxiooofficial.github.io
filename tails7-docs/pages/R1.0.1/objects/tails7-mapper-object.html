Tails7 Mapper Object

29 November 2025

Tails7 Mapper Object is responsible for operations such as: setting position, scaling, rotating, layering for ALL 3D.Objects Group objects when they are active.

This happens because this object has several behaviors that maintain a certain continuity.

We call this "Behavior chain mapping for 3D objects.Objects Group executable by Tails7 Mapper Object" (A very long name, I know), or simply "the chain".


Chain sequence

The chain sequence and order of subsequent behaviors:

  1. Distance Mode - Sets the distance between the target position of the 3D.Object Group and the reference point (mainly the camera).
  2. Mapping Mode - Calculations of the shader's inverse are applied to transform a given position through perspective to be adequate with the shader.
  3. Mapping Screen Mode - Displays the result of Mapping Mode but can also modify these results depending on the mode.
  4. Scale Mode - Scales the object to fit the calculations from Mapping Mode.
  5. Layer Mode - Layers each object individually to decide which should be on top and which underneath.
  6. Visible Mode - Determines when the object should be visible and when not.
