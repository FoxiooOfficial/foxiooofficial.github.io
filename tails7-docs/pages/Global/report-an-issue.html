Reporting Issues

All issues and bugs should be reported on our GitHub repository. When creating a new issue, please follow these guidelines:

  • Always include the version number in parentheses at the beginning of the title like: (R1.0.2) Error with object visibility
  • Provide detailed description of the issue
  • If possible, please provide a video, screenshots, or an .mfa example.
  • Add appropriate labels/tags from the table below

GitHub Issues: https://github.com/FoxiooOfficial/Tails7/issues

Note

Before reporting a problem, please ensure that you are using the latest version of Tails7 and Clickteam Fusion 2.5+.

Please also ensure that no one else has reported a similar problem.

Available Tags

When creating an issue, please use one of the following tags to categorize your report:

Tag Description
bug Something isn't working
documentation Improvements or additions to documentation
duplicate This issue or pull request already exists
enhancement New feature or request
good first issue Good for newcomers
help wanted Extra attention is needed
idea Your ideas that can be added
invalid This doesn't seem right
question Further information is requested
wontfix This will not be worked on
